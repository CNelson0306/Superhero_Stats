## I created and run this project using VS Code

# Superhero Browser using Python + Flask

# Features

Connects to the Superhero API to retrieve hero data  
Caches results using 'functools.lru_cache' to minimize API calls  
Flask web interface that:
Lists superheroes
Displays appearance details for a selected hero
Shows hero images
Clean project structure separating API logic from UI logic

# Requirements

Make sure you have:

Python 3.8+
pip or pip3
A valid Superhero API token  
Get one at: https://superheroapi.com/ use your GitHub sign in details.

# Installation

Copy or download this repository.

Install dependencies:

pip install flask requests

If using python3 explicitly (e.g., macOS):
pip3 install flask requests

# Running the Application

In the terminal from your project folder run the command:

python3 app.py (macOS)
python app.py
